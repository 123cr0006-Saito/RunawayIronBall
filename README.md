# 暴走鉄球  

プランナー２名 / プログラマー ２名 / CGデザイナー 5名 / キャラクターデザイナー 3名 / 声優 2名　　

計14名で制作したアクションゲームです。　　

制作期間 2023/11/24～2024/03/22　　

**概要説明**  
鉄球振り回しド派手アクション！！しゃべる鉄球と共にドンドン建物や敵をぶっ壊せ！！　　

# 開発環境
[DXライブラリ](https://dxlib.xsrv.jp/)  
[nlomann-json](https://github.com/nlohmann/json)
